## EnglishRead(1.0)
簡易版考單字軟體                                                                                            

自製題庫教學:
===========

* 1.在你要放題庫的地方建立一個txt檔(名字隨意)                                                                                            
                                                                                                    
* 2.打開txt檔                                                                                                    
                                                                                                    
* 3.在第一行寫EnglishReadUSE，大小寫有分                                                                                                   
                                                                                                    
* 4.在第二行寫題數，基本上如果是新的就寫0就好                                                                                                
                                                                                                    
* 5.完成創建題庫                                                                                                    
                                                                                                    
--------
題庫範例:                                                                                                    
EnglishReadUSE                                                                                                    
0                                                                                                    
1                                                                                                    
哈摟                                                                                                    
Hello                                                                                                    
                                                                                                    
2                                                                                                    
嗨                                                                                                    
Hi
